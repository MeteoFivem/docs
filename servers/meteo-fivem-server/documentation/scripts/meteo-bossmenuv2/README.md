# Meteo Boss Menu

This is a guide about testing the meteo fivem boss menu script designed exclusively for meteo server. This bossmenu comes with player application support ingame and more advanced features.

{% hint style="info" %}
Get access to our exclusive video testing guide on Discord to see all of this in action.
{% endhint %}

***

## Before You Start

* Access to the meteo test server
* Familiar with [getting started](../../getting-started.md) basics

***

## Testing Boss Menu

### Police Boss Menu

{% stepper %}
{% step %}
First use `/setjob yourid police 4`
{% endstep %}

{% step %}
Go to `/tp -558.51, -125.61, 43.89` to go to police bossmenu location
{% endstep %}

{% step %}
You can point target and open the police bossmenu
{% endstep %}

{% step %}
You can hire, fire players, add society money and do all those things
{% endstep %}

{% step %}
Also this society money is connected to society bank account on [meteo-banking](../meteo-banking/) which only can access by boss
{% endstep %}

{% step %}
Also all logs are saved on the boss menu too
{% endstep %}
{% endstepper %}

### Job Applications

{% stepper %}
{% step %}
For applications boss can enable or disable them
{% endstep %}

{% step %}
Go to `/tp -558.4874, -127.9775, 38` to check the application ped
{% endstep %}

{% step %}
Btw `/setjob` to something else like ambulance since if you already have the job you cant submit job application
{% endstep %}

{% step %}
Applicants get phone notifications when accepted or rejected
{% endstep %}
{% endstepper %}

### Other Job Boss Menu Locations
