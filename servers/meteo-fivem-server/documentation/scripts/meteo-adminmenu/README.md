## Testing Admin Menu

### Opening

* Keybind default is **F9**. you can use F9 as shortcut to open the menu or use `/admin` on the chat

### Features

* **Player management** - manage players, teleport, kick, ban, set jobs, give money, give items
* **Vehicle spawning** - spawn vehicles, customize, delete vehicles
* **Item spawning** - search and spawn any item to any player
* **Clothing/wardrobe** - access wardrobe and outfit management
* **Developer tools** - coordinate copying with built in keybind

### Developer Helper

* Our admin menu comes with the developer helper
* When you changing locations you can easily copy them with built in tool
* Use **F10** for noclip to fly around
* Use **F11** to copy coordinates. on developer menu you can choose how you want to copy coords and it will copy exactly like that format
