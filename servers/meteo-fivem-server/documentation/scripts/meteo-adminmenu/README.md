# Meteo Admin Menu

This is a guide about testing the meteo fivem admin menu script designed exclusively for meteo server.

{% hint style="info" %}
get access to our exclusive video testing guide on Discord to see all of this in action.
{% endhint %}

***

## Before You Start

* Access to the meteo test server
* Familiar with [getting started](../../getting-started.md) basics

***

## Testing Admin Menu

### Opening

* keybind default is **F9**. you can use F9 as shortcut to open the menu or use `/admin` on the chat

### Features

* **player management** - manage players, teleport, kick, ban, set jobs, give money, give items
* **vehicle spawning** - spawn vehicles, customize, delete vehicles
* **item spawning** - search and spawn any item to any player
* **clothing/wardrobe** - access wardrobe and outfit management
* **developer tools** - coordinate copying with built in keybind

### Developer Helper

* our admin menu comes with the developer helper
* when you changing locations you can easily copy them with built in tool
* use **F10** for noclip to fly around
* use **F11** to copy coordinates. on developer menu you can choose how u want to copy coords and it will copy exactly like that format

***

## Good to Know

* all keybinds, max money transfer limits and admin permissions are configurable on our config. you can change them once you get the server. we will guide you :)
