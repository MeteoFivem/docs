# Meteo Banking

This is a guide about testing the meteo fivem banking script designed exclusively for meteo server.

{% hint style="info" %}
Get access to our exclusive video testing guide on Discord to see all of this in action.
{% endhint %}

***

## Before You Start

* Access to the meteo test server
* Familiar with [getting started](../../getting-started.md) basics

***

## Testing Banking

{% stepper %}
{% step %}
**Bank**

Go to any bank location and point target and open bank menu. you can create bank accounts (up to 5 per player, $250 creation fee). manage your accounts, deposit, withdraw and transfer money
{% endstep %}

{% step %}
**Bank Locations**

* **Downtown** - `/tp 149.46, -1040.5, 29.37`
* **Pillbox** - `/tp 314.23, -278.83, 54.17`
* **Del Perro** - `/tp -1212.98, -330.78, 37.79`
* **Paleto** - `/tp -2962.47, 482.93, 15.70`
* **North Yankton** - `/tp -112.22, 6469.91, 31.63`
* **Blaine County** - `/tp 1175.07, 2706.63, 38.09`
{% endstep %}

{% step %}
**ATMs**

Also check ATMs around the map. they work the same way just point target and open. ATM models are at gas stations, banks and around the city
{% endstep %}

{% step %}
**Giving Cash**

And also you can point target to a player and give cash to them. this is part of banking
{% endstep %}
{% endstepper %}

> Make sure to check out the exclusive video for more details and see and check them out too

***
