# Test Commands

{% hint style="warning" %}
These commands are only available on our showcase server. use them to quickly test jail features without grinding.
{% endhint %}

***

## Available Commands

### /setjailrep \[playerID] \[amount]

Set a player's jail reputation to a specific value.

```
/setjailrep 1 500
```

| Parameter    | Description                       |
| ------------ | --------------------------------- |
| **playerID** | Server ID of the target player    |
| **amount**   | Reputation value to set           |

Skip the grind and test reputation-locked job tiers and rewards directly.

### /testalarm

Toggle the prison alarm on or off.

```
/testalarm
```

Run it once to activate, run it again to deactivate.

### /testmeal

Trigger a meal notification for all inmates.

```
/testmeal
```

Simulates a meal event so you can test the canteen and food collection flow.
