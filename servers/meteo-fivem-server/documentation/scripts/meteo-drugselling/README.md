## Before You Start

* Access to the meteo test server
* Know how to [spawn items](../../how-to-spawn-items.md)
* Familiar with [getting started](../../getting-started.md) basics

***

## Testing Drug Selling

### Alrighttt..

* Now make sure you have `cokebaggy`, `meth`, `joint`, `weed_brick` - any of these items. If you know what im talking about if you already did the [meteo-drugs](../meteo-drugs/) script testing guide.

{% hint style="warning" %}
Make sure to check the [meteo-drugs](../meteo-drugs/) testing guide if you haven't
{% endhint %}

### Selling Drugs

{% stepper %}
{% step %}
### Get Drugs to Sell

Lets spawn `weed_brick`. You can use /giveitem or admin menu to spawn
{% endstep %}

{% step %}
### Offer Drugs to NPC

Find NPC and now when you point target on them it will show offer drugs new option. Click on that and offer drugs
{% endstep %}

{% step %}
### Set Price

After that use **arrow left** and **right** to adjust a price and press **E** to confirm
{% endstep %}

{% step %}
### Collect Cash

After you sell you will get folded cash :)
{% endstep %}

{% step %}
### Wash Money

Folded cash can be washed using doing labor jobs. Check out those job guides
{% endstep %}
{% endstepper %}

> Also check out [meteo-perks](../meteo-perks/) testing guide since money washing rate is connected to your crime level

### Organization Territory

* If you are in an organization and selling drugs on your org territory you will gain more XP for your org and earn more cash
* Check out [meteo-organizations](../meteo-organizations/) testing guide for more info about territories

***

## Good to Know

{% hint style="success" %}
All drug prices, sell cooldowns, NPC accept chances and territory bonuses are configurable on our config. You can change them once you get the server. We will guide you :)
{% endhint %}

> Connected with [meteo-perks](../meteo-perks/). If you have **Hustler** specialization - Smooth Talker increases sell prices, Connections reduces sell cooldowns, Pure Product makes product sell for more, Turf Control gives +70% org rep from drug sales
