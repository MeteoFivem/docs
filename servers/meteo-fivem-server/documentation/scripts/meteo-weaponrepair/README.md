# Meteo Weapon Repair

This is a guide about testing the meteo fivem weapon repair script designed exclusively for meteo server.

{% hint style="info" %}
get access to our exclusive video testing guide on Discord to see all of this in action.
{% endhint %}

***

## Before You Start

* Access to the meteo test server
* Know how to [spawn items](../../how-to-spawn-items.md)
* Familiar with [getting started](../../getting-started.md) basics

***

## Testing Weapon Repair

### Step 1 - Get a Weapon and Damage It

* get weapon using `/giveitem yourid weapon_pistol_mk2 1` or use meteo admin menu
* use the weapon till its health is lower
* you can see the weapon health on inventory when hovering on weapon item

### Step 2 - Go to Repair Location

* use `/tp 2409.2043, 3031.5886, 48.1526` on chat to teleport there

### Step 3 - Store the Weapon

* point target at the repair bench and open storage
* put the weapon to that slot (you can repair 2 weapons at a time - this is configurable)

### Step 4 - Check Repair Cost

* point target and use repair weapon option
* this will show what materials and cost you need to repair the weapon
* repair cost is $500 per weapon
* materials needed: plastic, metalscrap, copper, aluminum, steel, rubber (2 to 4 items needed depending on damage)

### Step 5 - Get the Materials

* usually they are materials which players can find on scrap and dumpsters like doing those activities
* you can open admin menu using **F9** and spawn those items
* or use commands like `/giveitem yourid metalscrap 5`, `/giveitem yourid plastic 5` etc.

### Step 6 - Repair

* after that confirm repair and do the minigame and finish it
* the skill check has 3 rounds with mixed difficulty (easy, medium, hard)
* if you fail you keep your materials - just try again
* finally open storage again and get the repaired weapon

***

## Good to Know

* all repair costs, materials, skill check difficulty and locations are configurable on our config. you can change them once you get the server. we will guide you :)
* repair bench location has an optional map blip that can be enabled in config
