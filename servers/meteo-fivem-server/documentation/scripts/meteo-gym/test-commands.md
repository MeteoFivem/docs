# Test Commands

{% hint style="warning" %}
These commands are only available on our test server. use them to quickly test gym features without grinding.
{% endhint %}

***

## Available Commands

### /setgymstats

Max out all gym stats instantly. sets strength, dexterity, endurance, and energy to maximum.

```
/setgymstats
```

### /gym \[action] \[stat] \[value]

Fine-grained control over gym stats.

#### Set a specific stat

```
/gym set strength 500
/gym set dexterity 800
/gym set endurance 300
```

Stats: `strength`, `dexterity`, `endurance`

#### Add to a stat

```
/gym add strength 100
```

#### Set energy

```
/gym energy 250
```

#### Reset all stats

```
/gym reset
```

#### Max all stats

```
/gym max
```

#### View current stats

```
/gym info
```

Prints stats to F8 console.

### /cleargyminjury

Quick way to clear your active gym injury.

```
/cleargyminjury
```
