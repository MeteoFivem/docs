# Meteo Report Menu

This is a guide about testing the meteo fivem report script designed exclusively for meteo server.

{% hint style="info" %}
get access to our exclusive video testing guide on Discord to see all of this in action.
{% endhint %}

***

## Before You Start

* Access to the meteo test server
* Familiar with [getting started](../../getting-started.md) basics

***

## Testing Reports

* you can use report menu using `/report` or using f1 menu > general > report
* select a report type - there are player reports (RDM, VDM, metagaming, exploiting etc.), bug reports, suggestions and questions
* fill out the details and submit
* admins will automatically get access to dashboard and manage reports and review them
* reports have priority levels (low, medium, high, urgent) based on category
* admins can filter by status - pending, investigating, resolved

### Report Categories

**player reports:** RDM, VDM, metagaming, powergaming, fail RP, griefing, exploiting, harassment, other

**general reports:** bug report, suggestion, server issue, mapping issue, economy issue, job related, other

**questions:** general question, rules clarification, jobs and economy, vehicles, properties, commands, technical support, other

***

## Good to Know

* the report command name, categories and priority levels are configurable on our config. you can change them once you get the server. we will guide you :)
