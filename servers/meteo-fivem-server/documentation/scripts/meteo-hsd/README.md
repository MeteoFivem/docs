# Meteo High Speed Drop (HSD)

This is a guide about testing the meteo fivem high speed delivery script designed exclusively for meteo server.

{% hint style="info" %}
Get access to our exclusive video testing guide on Discord to see all of this in action.
{% endhint %}

{% hint style="warning" %}
Make sure to follow the [meteo-crimetablet](../meteo-crimetablet/) guide before following this.
{% endhint %}

***

## Before You Start

* Access to the meteo test server
* Familiar with [getting started](../../getting-started.md) basics

***

## Testing High Speed Drop

### Purchasing

{% stepper %}
{% step %}
Open tablet and go to services app. you can see the High Speed Drop service
{% endstep %}

{% step %}
If you do not have crypto use `/addcrypto 200` and get crypto

{% hint style="warning" %}
This is a testing command and only works on our testing server
{% endhint %}
{% endstep %}

{% step %}
Purchase it (costs 150 MTC) and start
{% endstep %}

{% step %}
You can do this with a group too. follow group guide on [crimetablet](../meteo-crimetablet/) guide and add group members too
{% endstep %}
{% endstepper %}

### Doing the Delivery

{% stepper %}
{% step %}
Go to the marked location on the map. also if you are doing this with group they will see the marker too
{% endstep %}

{% step %}
Point target and interact with ped and collect the package
{% endstep %}

{% step %}
Then it will start with giving you a package and also a vehicle. after that you have to deliver that package as fast as possible to the locations. make sure the vehicle is near. otherwise you cant deliver it without the vehicle :)
{% endstep %}

{% step %}
Repeat and complete all 6 delivery locations and finish the delivery
{% endstep %}

{% step %}
Then drop the vehicle at the dropoff location and finish and get rewards
{% endstep %}
{% endstepper %}

### Rewards

* Fastest players will get golden rewards (300 MTC, reward boxes, chop contracts)
* Others will get normal rewards (160 MTC)
* Personal and global leaderboard tracks your best times
* 10 minute time limit to complete all deliveries
