# Meteo Apartments

This is a guide about testing the meteo fivem apartments script designed exclusively for meteo server.

Get access to our exclusive video testing guide on Discord to see all of this in action.
{% endhint %}

***

## Before You Start

* Access to the meteo test server
* Know how to [spawn items](../../how-to-spawn-items.md)
* Familiar with [getting started](../../getting-started.md) basics

***

## Testing Apartments

{% stepper %}
{% step %}
**Getting Started**

We already have 210+ rooms created for you. so you dont have to create them again. and also you can add more complexes if you need more. we have WiWang Hotel. just use `/tp -824.7368, -702.1234, 28.0600` to go there. and interact with ped to see all available apartments. you can even purchase new ones or just see your current owned ones and sell them from here too

{% hint style="warning" %}
If you dont know how to get money use `/setmoney yourid bank 100000` to get money. this is a testing command.
{% endhint %}
{% endstep %}

{% step %}
**Buying and Entering Apartment**

Lets go to your apartment floor if you have one already or lets just buy and start with new one. room types: Classic ($35,000), Modern ($50,000), Luxury Penthouse ($150,000). go near the door and you will see lock sprite and you can use e to open but if its not owned then it will show purchase menu and you can purchase it
{% endstep %}

{% step %}
**Furnishing and Managing**

After purchasing open furnishing menu using f1 menu and place the tablet to access the property menu. you can manage the apartment from here. and also give access, and go upgrades etc etc
{% endstep %}

{% step %}
**Security and Lockpicking**

Lets check the door lock protection. also other players can rob these apartments (only if they are online and also can config that too). go near the apartment door after it locked and use lockpick and break the apartment door and unlock it. also make sure you have the police job to see the police alerts. also if you have a friend you can rob his apartment and also can rob his storages and get items. also upgrade your security system and see if its going harder when lockpick. security levels go from Level 1 (Basic, free) up to Level 6 (Maximum Security, $100,000). also this lockpick script is connected to our [meteo-perks](../meteo-perks/). when player have perks they can do the lockpick more easily
{% endstep %}
{% endstepper %}

{% hint style="warning" %}
This script is same on the meteo-properties too. make sure to check [that](../meteo-properties/) also
{% endhint %}
