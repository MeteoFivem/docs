# Meteo Police Job

This is a guide about testing the meteo fivem police job script designed exclusively for meteo server.

{% hint style="info" %}
get access to our exclusive video testing guide on Discord to see all of this in action.
{% endhint %}

***

## Before You Start

* Access to the meteo test server
* Police job: `/setjob yourid police 4`
* Familiar with [getting started](../../getting-started.md) basics

***

## Testing Police Job

* testing guide coming soon. please check out the exclusive video testing guide on Discord for now
* make sure to also check these connected scripts:
  * [meteo-dispatch](../meteo-dispatch/) for 911 calls and alerts
  * [meteo-mdt](../meteo-mdt/) for mobile data terminal
  * [meteo-evidence](../meteo-evidence/) for evidence collection
  * [meteo-fingerscanner](../meteo-fingerscanner/) for fingerprint scanning
  * [meteo-policeradar](../meteo-policeradar/) for speed radar
  * [meteo-jobgarage](../meteo-jobgarage/) for police vehicles
  * [meteo-jail](../meteo-jail/) for jailing players

***

## Good to Know

* all police settings, locations, armory items and permissions are configurable on our config. you can change them once you get the server. we will guide you :)
