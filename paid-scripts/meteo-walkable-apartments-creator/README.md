### Features

* In-Game Creator - Point and click to configure doors, zones, and rooms (no other apartment script has this)
* Apartment raids and lockpick support with storage item steal
* Furniture Support - Full `meteo-furnishing` integration for customizable interiors
* Multi-Door Support
* Fully Optimized with zone and point based hybrid system



***

### What You Can Do:

* Take any apartment MLO and turn it into functional apartments
* Add new rooms using the creator tool
* Let players furnish their own apartments
* Upgrade door security, enable lockpicking and police raids
* Multi-floor elevator navigation built-in

No teleporting. No fake instances. Just real walkable apartments in FiveM.
