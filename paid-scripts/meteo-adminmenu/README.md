# Meteo Admin Menu

***

### What is Meteo Admin Menu?

A complete in-game admin menu for FiveM. Replaces command-spam admin workflows with a clean UI for player management, vehicle spawning, server controls, dev tools, and audit logging.

Drop-in replacement for qb-adminmenu/qbx_adminmenu and similar admin scripts.

***

### Features

| Feature                  | Description                                                                |
| ------------------------ | -------------------------------------------------------------------------- |
| **Player Management**    | Kick, ban, freeze, revive, spectate, teleport, message                     |
| **Spectate Mode**        | One-click spectate with Backspace exit and on-screen keybind hint          |
| **Vehicle Tools**        | Spawn, save, delete, fix, max mods, customization menu hook                |
| **Money Management**     | Give cash/bank with configurable min/max limits                            |
| **Server Controls**      | Change weather, time, announcements                                        |
| **Dev Tools**            | Copy coords (vec2/vec3/vec4), noclip, blip toggle, entity free view        |
| **Permission Tiers**     | `admin` and `god` levels via QBCore permissions or ace principals          |
| **Audit Logs**           | All actions logged to Discord webhook or ox\_lib logger                    |
| **Configurable Keybinds**| Default `F9` admin menu, `F10` noclip, `F11` coords - rebindable in-game   |
| **Multi-Language**       | Full ox\_lib locale support                                                |

***

### Requirements

* QBCore or QBox
* ox\_lib
* ox\_inventory
* oxmysql
* meteo-keybinddisplay
