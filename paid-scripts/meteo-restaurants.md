# Meteo Restaurants

***

### What is Meteo Restaurants?

A restaurant script for FiveM. Create unlimited restaurants with employees, multi-step cooking, orders, banking, and customer shop - all managed in-game.

Works with both **QBCore** and **QBox**.

***

### Features

| Feature                     | Description                                          |
| --------------------------- | ---------------------------------------------------- |
| **Multi-Restaurant**        | Unlimited restaurants, each with their own settings   |
| **Employees**               | Hire, fire, grades, paychecks, suspensions            |
| **Cooking**                 | Multi-step recipes with process, cook, and drink stations |
| **Orders**                  | Create orders, track status, pay anytime              |
| **Banking**                 | Restaurant bank, deposits, withdrawals, logs          |
| **Commissions**             | Employee commissions based on performance             |
| **Shop**                    | Direct customer purchases from shop stash             |
| **Doorlocks**               | Built-in door management per restaurant               |
| **Work Zones**              | Auto clock-out when leaving restaurant area           |
| **Paychecks**               | Auto paychecks for clocked-in employees               |
| **50+ Unique Items**        | Custom food and drinks with props and animations      |
| **Multi-Language**          | Full ox\_lib locale support                           |
| **Phone**                   | Notifications and banking via phone                   |
| **Applications**            | Players can apply for jobs at restaurants              |
| **In-Game Admin**           | Create and manage restaurants without config editing   |

***

### Requirements

* QBCore or QBox
* ox\_lib
* oxmysql
* ox\_target or qb-target
* meteo-keybinddisplay
